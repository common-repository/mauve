MAUVE++ allows you to check the accessibility of WordPress contents while editing them and before you hit publish. In-post accessibility scanner and guidance for WCAG 2.1 compliance.

== Description ==

Enhance the accessibility of your WordPress website effortlessly with MAUVE++, the accessibility evaluation plugin. This tool is designed to assist website owners, developers, and content creators in ensuring that their digital content complies with the Web Content Accessibility Guidelines (WCAG) 2.1 standards, making it more inclusive for all users, including those with disabilities.

### Key Features

- Accessibility Checks: once the evaluation start button is clicked MAUVE++ checks your website\'s content,  to identify potential accessibility issues, and generates detailed summaries. It provides a list of accessibility errors and warning  on your post/page edit screen and allows you to scan your content even after its updating.

- Real-time Evaluation Reports: the plugin provides instant feedback on your page/post accessibility with real-time evaluation reports. For each error or warning flagged, MAUVE++ indicates the violated success criterion, the technique used, the number of occurrences of the error/warning and the documentation needed to solve the issue. It also provides the code of the element that triggered the error/warning and allows you to highlight it within the page or view it in the source code.

- WCAG 2.1 Compliance: MAUVE++ ensures that your contents are aligned with the latest Web Content Accessibility Guidelines (WCAG) 2.1, fostering an inclusive online experience.

- User-Friendly Interface: the plugin features a user-friendly interface, making it accessible to both developers and non-technical users. Non-technical users can navigate through the evaluation results, view detailed explanations and identify the issue within the page by highlighting it with a click.

- Developer-Friendly Tools: for developers, MAUVE++ offers tools for viewing and analyzing the accessibility details within the source code. Dive into the developer view to inspect issues at a code level and facilitate efficient debugging.

** MAUVE++ works only with Gutenberg editor **


== Service Reliance ==

MAUVE++ relies on a 3rd-party service for some of its functionalities. The accessibility evaluation and real-time reports are powered by MAUVE++ validator, a trusted accessibility evaluation service provided by the MAUVE++ project.

### MAUVE++ Validator Service

MAUVE++ utilizes the MAUVE++ validator service to perform accessibility evaluations in real-time. This service analyzes website content to identify potential accessibility issues and provides detailed reports to help users ensure their digital content complies with Web Content Accessibility Guidelines (WCAG) 2.1 standards.

- Service Homepage: [MAUVE++ Validator](https://mauve.isti.cnr.it/index.jsp)
- Privacy & Cookie Policy: [MAUVE++ Validator Privacy & Cookie Policy](https://mauve.isti.cnr.it/privacy.jsp)

Additionally, the plugin provides links to relevant resources on the W3C website, offering detailed explanations on how to satisfy the techniques used in the accessibility evaluation. Users can access these resources to gain further insight into accessibility guidelines and techniques.

- W3C Accessibility Techniques: [W3C Accessibility Techniques](https://www.w3.org/WAI/WCAG21/Techniques/). Copyright Â© 2023 World Wide Web Consortium. https://www.w3.org/copyright/document-license-2023/

== Installation ==

- Upload the MAUVE++ folder to the /wp-content/plugins/ directory, or effortlessly install the plugin directly through the \'Plugins\' screen in WordPress.

or

- Activate the plugin by navigating to the \'Plugins\' screen in WordPress.
